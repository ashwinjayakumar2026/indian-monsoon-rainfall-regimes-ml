The IMD rainfall dataset used in this project is not included due to licensing restrictions.

Users can obtain the IMD 0.25° gridded rainfall dataset from the India Meteorological Department (IMD) and update the file path in the notebook.

The India country boundary shapefile used for visualization is also not included. Users can obtain an India administrative boundary shapefile from publicly available geospatial data sources such as Natural Earth or GADM and update the path in the notebook.
