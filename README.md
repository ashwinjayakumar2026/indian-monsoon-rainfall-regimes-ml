This project applies unsupervised machine learning (K-Means clustering) to identify spatial rainfall regimes across India using the IMD 0.25° gridded rainfall dataset (1979–2022).

## Methodology

1. Extract JJAS monsoon rainfall from IMD dataset
2. Compute rainfall statistics:
   - Mean rainfall
   - Standard deviation
   - 95th percentile rainfall
   - Wet day frequency
3. Standardize features
4. Apply K-Means clustering
5. Visualize rainfall regimes across India
